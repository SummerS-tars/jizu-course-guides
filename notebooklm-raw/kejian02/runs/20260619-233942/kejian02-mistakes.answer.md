根据课件 02 及相关复习资料，以下是计算机组成中 5 组最易混淆的核心概念对比说明：

### 1. 定点数编码：原码 vs 补码 vs 移码
这三种编码主要解决符号位处理及运算简化问题 [1, 2]。

| 特性 | 原码 (Sign and Magnitude) | 补码 (Two's Complement) | 移码 (Excess/Biased) |
| :--- | :--- | :--- | :--- |
| **定义** | 符号位 + 数值绝对值 [3, 4] | 正数同原码；负数各位取反末位加 1 [5, 6] | 真值加上一个偏置常数 (Bias) [7, 8] |
| **0 的表示** | 不唯一（+0 和 -0）[3, 9] | 唯一（均为全 0）[6, 10, 11] | 唯一 [12, 13] |
| **运算特点** | 符号位需单独处理，加减法不统一 [3, 11] | 符号位参与运算，减法可统一为加法 [10, 11] | 符号位与真值符号相反，便于阶码比较 [7, 14] |
| **主要用途** | 现用于表示浮点数的尾数 [10, 15] | 现代计算机表示定点整数的标准 [10, 11] | 表示浮点数的阶码 [8, 12, 14] |

**来源：** 课件 02 [1, 3, 7]；Week 5 [16, 17]。

---

### 2. 浮点数类型：规格化 vs 非规格化数
IEEE 754 标准通过这两种形式扩展了实数的表示范围 [18, 19]。

| 特性 | 规格化数 (Normalized) | 非规格化数 (Denorms) |
| :--- | :--- | :--- |
| **阶码 (E) 特征** | 非全 0 且非全 1 (1 ~ 254) [18, 20] | 全 0 (E = 0) [18, 19, 21] |
| **隐藏位 (隐含位)** | 默认小数点前为 1 (1.f) [22-24] | 默认小数点前为 0 (0.f) [19, 25] |
| **数值计算** | $(-1)^S \times 1.f \times 2^{E-127}$ [20, 23] | $(-1)^S \times 0.f \times 2^{-126}$ [19, 25] |
| **作用** | 表示常规范围内的浮点数 [18, 26] | 填补 0 与最小规格化数间的间隙（逐渐下溢）[19, 25, 27] |

**来源：** 课件 02 [18, 21, 25]；Week 5 [17]。

---

### 3. 特殊非数：SNaN vs QNaN
NaN（非数）用于处理非法运算，分为“发信号”和“静默”两种 [26, 28]。

| 特性 | 发信号非数 (SNaN) | 静默非数 (QNaN) |
| :--- | :--- | :--- |
| **尾数最高位** | 通常为 0（且尾数其余位不全为 0）[29] | 通常为 1 [29] |
| **异常触发** | 作为操作数时会触发浮点异常 [26, 30] | 参与运算不触发异常，结果仍为 QNaN [26, 28] |
| **设计目的** | 捕捉未初始化的变量或非法操作 [26, 29] | 允许程序在发生计算错误时继续执行 [26, 28, 29] |

**来源：** 课件 02 [18, 28]；Week 5 [17]。

---

### 4. 字节序：大端 vs 小端
字节序决定了多字节数据在内存中的排列顺序 [31, 32]。

| 特性 | 大端方式 (Big Endian) | 小端方式 (Little Endian) |
| :--- | :--- | :--- |
| **存储原则** | 最高有效字节 (MSB) 存放在低地址 [33-35] | 最低有效字节 (LSB) 存放在低地址 [33-35] |
| **直观感受** | 与人类书写习惯一致（高位在前）[36] | 逻辑地址增加时，数据权重随之增加 [37] |
| **典型代表** | IBM 360/370, MIPS (旧), 网络字节序 [33, 34] | Intel x86, RISC-V [33, 34, 38] |

**来源：** 课件 02 [31, 33, 39]；Week 16 复习 [40, 41]。

---

### 5. 浮点格式：单精度 vs 双精度位域
IEEE 754 定义了不同宽度的位域以平衡范围和精度 [23, 24]。

| 位域组成 | 单精度 (float / FP32) | 双精度 (double / FP64) |
| :--- | :--- | :--- |
| **总位数** | 32 位 [23, 24, 42] | 64 位 [23, 24, 42] |
| **符号位 (S)** | 1 位 [20, 23] | 1 位 [20, 23] |
| **阶码 (E)** | 8 位（偏置值 127）[17, 20, 23] | 11 位（偏置值 1023）[17, 20, 23] |
| **尾数 (f)** | 23 位（有效位 24 位）[16, 20, 23] | 52 位（有效位 53 位）[17, 20, 23] |
| **精度** | 约 7 位十进制有效数字 [43, 44] | 约 17 位十进制有效数字 [43, 44] |

**来源：** 课件 02 [23, 31, 42]；Week 5 [16, 17]；Week 16 复习 [20, 45]。
