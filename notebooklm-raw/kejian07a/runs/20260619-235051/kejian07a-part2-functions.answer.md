根据课件 7a 和 Week14 课程记录，互连函数用于描述网络节点间的映射关系，以下是相关定义与手算示例：

### 1. 互连函数定义

*   **Cube（立方体交换函数）**：将二进制编号的第 $k$ 位取反，记作 $Cube_k(x_{n-1}...x_k...x_0) = x_{n-1}...\bar{x}_k...x_0$ [1, 2]。
*   **Shuffle（均匀洗牌函数）**：将二进制编号循环左移一位，即将最高位移至最低位，记作 $S(x_{n-1}x_{n-2}...x_1x_0) = x_{n-2}...x_1x_0x_{n-1}$ [1, 3]。
*   **Butterfly（蝶式函数）**：将二进制编号的最高位与最低位互换位置，记作 $B(x_{n-1}x_{n-2}...x_1x_0) = x_0x_{n-2}...x_1x_{n-1}$ [3, 4]。
*   **反位序函数**：将二进制编号的位序完全颠倒，记作 $R(x_{n-1}x_{n-2}...x_0) = x_0x_1...x_{n-1}$ [4]。
*   **PM2I（加减 $2^i$ 函数）**：将输入端编号循环移动 $2^i$ 个位置，表达式为 $PM2_{\pm i}(X) = (X \pm 2^i) \mod N$ [3, 4]。

### 2. 数值例：3 位二进制映射手算

假定源地址为 **$X = 6$**（二进制 **$110_2$**），计算其映射结果：

#### **Shuffle 映射计算过程：**
1.  写出源地址二进制：$110$ [1]。
2.  执行循环左移操作：将最左边的 $1$ 移到最后，其余位前移 [3]。
3.  得到新序列：$101$ [1]。
4.  **结果**：$Shuffle(110) = 101_2$（十进制 **$5$**）[1, 3]。

#### **Butterfly 映射计算过程：**
1.  写出源地址二进制：$110$ [4]。
2.  识别位位置：最高位是 $1$，最低位是 $0$ [3]。
3.  交换最高位与最低位：$0$ 换到最左边，$1$ 换到最右边，中间位不变 [3]。
4.  得到新序列：$011$ [4]。
5.  **结果**：$Butterfly(110) = 011_2$（十进制 **$3$**）[3, 4]。
