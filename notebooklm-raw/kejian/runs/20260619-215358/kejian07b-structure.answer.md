根据课件 7b《层次结构存储系统》及相关课堂、实验记录，该课件的内部结构梳理如下：

### 1. 章节/板块原始顺序
*   **存储器概述和RAM芯片**：存储元 (Cell)、编址单位、存储阵列 [1]。
*   **主存与CPU的连接及读写**：主存模块连接、“装入”(Load) 和 “存储”(Store) 指令的操作流程 [1, 2]。
*   **高速缓冲存储器 (Cache)**：程序访问局部性（时间/空间）、工作原理、Cache 行与主存块映射方式、Cache RTL 实现、Cache 与程序性能 [1, 3]。
*   **虚拟存储器 (Virtual Memory)**：虚拟地址空间、SV39 地址格式、页表项 (PTE) 位定义、satp 寄存器、TLB 原理、地址转换实例 [1, 3, 4]。

### 2. 知识点维度分 Part
*   **Part 1：存储物理与接口层**：涵盖 RAM 芯片原理及 Load/Store 指令在总线上的数据流动 [1, 2]。
*   **Part 2：缓存加速层**：涵盖 Cache 的直接映射、组相联、全相联逻辑及 AMAT 性能评估 [1, 5, 6]。
*   **Part 3：特权与地址转换层**：涵盖 **Sv39** 规范、**SATP** 字段、**TLB** 缓存及多级页表遍历机制 [4, 7]。

### 3. 重要程度标注
*   **Cache 映射与 AMAT 计算**：**期末重点**，需掌握不同映射下的手算模拟 [6, 8]。
*   **Sv39 / SATP / TLB**：**期末核心**，涉及地址转换全过程及特权级控制，是期末复习的核心板块 [6, 9]。

### 4. 课堂对照与 Lab 对应关系
*   **课堂对照 (Week 10–12)**：
    *   Week 10 深入讲解了存储器实验及 Cache 状态机实现 [10]。
    *   Week 11 重点剖析了 **SATP** 寄存器的 WARL 机制及 **Sv39** 三级页表结构 [7]。
*   **Lab 对应关系**：
    *   **Lab 4**：实现 CSR 指令及 satp 等特权寄存器 [11, 12]。
    *   **Lab 5**：实现 **MMU**，完成基于 **Sv39** 的三级页表遍历地址翻译 [13-15]。
    *   **Lab 6**：处理异常与中断，涉及异常引起的数据通路 flush [15, 16]。

### 5. 关键术语与参考书对应
*   **关键术语**：局部性原理 (Locality)、PTE (V/R/W/X/A/D 位)、ASID、TLB Hit/Miss、Page Fault [4, 5, 17, 18]。
*   **参考书对应**：
    *   **唐朔飞**：第 4 章 存储器 [19]。
    *   **P&H (RISC-V版)**：第 5 章 Large and Fast: Exploiting Memory Hierarchy [20, 21]。
