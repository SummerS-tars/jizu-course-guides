根据**课件 03**（Part 2：定点加减法），为您整理补码运算与标志位判断的知识点及手算示例：

### 1. 补码加减、求负规则
*   **加法规则**：$[x+y]_补 = [x]_补 + [y]_补 \pmod{2^n}$。符号位与数值位一起参与运算，符号位产生的进位直接丢弃 [1-3]。
*   **减法规则**：$[x-y]_补 = [x]_补 + [-y]_补 \pmod{2^n}$。通过将减法转换为加法实现 [1-3]。
*   **求负（求补）规则**：已知 $[y]_补$，求 $[-y]_补$ 的方法是：**“各位取反，末位加 1”**（包括符号位）[1, 3, 4]。

### 2. 标志位 OF/SF/ZF/CF 产生条件
这些标志位由硬件电路根据运算结果自动生成 [5, 6]：
*   **ZF (Zero Flag)**：零标志。若运算结果 $F = 0$，则 $ZF=1$；否则为 0 [5-7]。
*   **SF (Sign Flag)**：符号标志。等于运算结果的最高位（即 $F_{n-1}$）[5-7]。
*   **OF (Overflow Flag)**：溢出标志（仅对**有符号数**有效）。
    *   逻辑判断：$OF = C_n \oplus C_{n-1}$（即最高位产生的进位与次高位产生的进位异或）[5, 8]。
    *   直观判断：若两个同号数相加，结果符号与加数相反，则溢出 [9-11]。
*   **CF (Carry Flag)**：进/借位标志（仅对**无符号数**有效）。
    *   逻辑判断：$CF = C_{out} \oplus Sub$（$Sub=0$ 表示加法，$Sub=1$ 表示减法）[5, 8, 10]。
    *   加法：最高位有进位则 $CF=1$。
    *   减法：无进位（需借位）则 $CF=1$ [10, 12]。

### 3. 数值例：8 位补码加法手算
**题目**：给定 8 位补码 $[x]_补 = 0110\,1011$ (107) 和 $[y]_补 = 0010\,1110$ (46)，计算 $x+y$ 并判断标志位 [9]。

**逐步计算过程**：
1.  **列竖式相加**：
    ```text
       次高位进位 C7=1
       ↓
       0110 1011  (107)
    +  0010 1110  (46)
    ------------
     0 1001 1001  (结果 F)
     ↑
    最高位向外进位 C8=0
    ```
2.  **判断标志位**：
    *   **ZF = 0**：结果为 `1001 1001`，不为 0 [9]。
    *   **SF = 1**：结果最高位为 1 [9]。
    *   **OF = 1**：$C_8(0) \oplus C_7(1) = 1$。从真值看，两个正数相加得到了负数（-103），发生溢出 [9, 13]。
    *   **CF = 0**：加法运算且最高位无进位（$0 \oplus 0 = 0$）[9]。

这个例子说明，在带符号数运算中，由于 $OF=1$，该结果是错误的；但在无符号数运算中，由于 $CF=0$，结果 153 是正确的 [13]。
