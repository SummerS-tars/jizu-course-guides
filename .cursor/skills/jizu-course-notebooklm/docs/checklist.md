# 定稿检查清单

> 每份学习指南定稿前，Agent 必须逐项检查。

## 内容完整性

- [ ] 覆盖该模块全部周次的课程内容
- [ ] 与 `guides/计组课程-15周内容梳理.md` 进度同步
- [ ] 所有 batch 的 raw 答案已通读并整合
- [ ] 标注了 NotebookLM 与课纲的偏差（如有）

## 叙事质量

- [ ] 每个大模块有全景节（学什么 / 学完能做什么 / 自检问题）
- [ ] 章级叙事线 + 节级「本节要回答」+ 节间承接
- [ ] ≥3 张 mermaid 图（数据通路/流水线/存储层次等）
- [ ] ≥2 组易混对比表（如 RISC vs CISC、Cache 映射策略等）
- [ ] ≥3 处追问/直观理解块

## 技术准确

- [ ] 公式正确，符号一致
- [ ] RISC-V 汇编语法正确
- [ ] IEEE 754 表示计算示例验证无误
- [ ] 数据通路图与控制信号对应正确
- [ ] 标明了参考来源（课件编号/教材章节）

## 格式

- [ ] Markdown 排版正确
- [ ] Mermaid 语法可渲染
- [ ] 代码块标注语言（asm / verilog / python / bash）
