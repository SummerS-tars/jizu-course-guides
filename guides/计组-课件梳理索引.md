1. **授课顺序 ≠ 课件编号**：Week 1–3 先讲通路/ISA，Week 4–5 才回溯数据表示；复习时勿死跟课件序号。

4. **课件 5/7 拆分**：`5_指令级并行`（5b）、`7_互连网络`（7a）、`7_层次结构存储系统`（7b）各算独立梳理单元，共 **12 份**。

## 四、单份课件学习指南结构（模板）

每份 `guides/计组-课件{NN}-学习指南.md` 包含：

1. **元信息**：课件 PDF、NotebookLM source、课堂覆盖、唐朔飞 + P&H 章§、关联周次指南链接  
2. **课件内容覆盖索引**（★无遗漏）：discovery 中**每个课件板块**一行，映射到 `Part · 块`  
3. **Part 正文**（严格按课件原序，不按上课周）  
   - Part 内按**知识点维度分块**（块 X.Y），**不按页**  
   - 每块：**课件要点** → **讲解** → 重难点加 **直观理解 / 手算例 / mermaid**  
   - 涉及 Lab4–6 的 Part：标注 Wiki / `26-Arch/Doc/Lab*/report.md` 交叉点  
4. **术语表**、**复习优先级**、**与周次指南对照表**、**raw 路径**

标杆（周次轨）：`guides/计组-Week1-3-学习指南.md`  
试点骨架（课件轨）：`guides/计组-课件05-学习指南.md`
